# Product Focus (PMF)

## Target user
Local-first app builders who need embedded semantic search for notes, files, or offline-first
knowledge bases without running a separate vector database.

## Top 10 high-impact features (ranked)
1. Web console for managing tables, ingest, and search.
2. One-click demo dataset and guided quickstart.
3. Search results with row previews and embedding status.
4. Schema designer with templates and validation.
5. Bulk ingest (JSON/CSV) with progress feedback.
6. Background embedding worker with retries and backoff.
7. Hybrid keyword + vector search.
8. Portable snapshots/backup export.
9. Pluggable embedder config (local + optional remote).
10. Local replication/sync tooling.

## Shipped now
- Web console served by the HTTP server.
- Demo dataset flow with sensible defaults.
- Search + embedding controls (process jobs, stats, previews).
